# docs ディレクトリ

このディレクトリはコンペティションに関する情報を整理して管理します。

## ディレクトリ構成

### `official/`

Kaggle 公式から提供される情報を記載します。

- コンペティションの概要・ルール
- 評価指標の詳細
- データの説明
- 制約事項

**例**: `competition.md`

### `discussion/`

Kaggle Discussion から得られた情報を記載します。

- 他の参加者からの知見・Tips
- 有用なコード片やアプローチ
- バグ報告や注意点
- データに関する追加情報

**命名規則**: `YYYY-MM-DD_topic.md`

### `insights/`

自分の実験から得られた知見を記載します。

- 実験結果の考察
- 失敗した試みとその理由
- 有効だったテクニック
- 今後試すべきアイデア

**命名規則**: `YYYY-MM-DD_topic.md`
